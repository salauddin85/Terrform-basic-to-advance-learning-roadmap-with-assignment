# 🚀 Terraform Learning Roadmap (Basic → Advanced)

**Author:** Md. Salauddin  
**Role:** Software Developer, PEPOLTEK LTD  

---

## 📌 Overview

This roadmap is designed for an intern DevOps developer who already has basic
familiarity with AWS (EC2, SSH, S3 via console) and wants to become proficient
in Terraform from beginner to advanced level.

The training is structured into three progressive phases:

- Basic
- Intermediate
- Advanced

Each phase builds on the previous one through hands-on practice and real-world scenarios.

---

## 🗺️ Learning Flow

| Phase        | Topics Covered | Assignment |
|-------------|--------------|------------|
| **Basic** | Terraform setup, first config, state, variables, outputs, CLI | Launch EC2 with SSH, output public IP |
| **Intermediate** | Remote state, modules, data sources, count/for_each, functions, workspaces | Web server module with dev/prod |
| **Advanced** | Terraform Cloud, state import, lifecycle, CI/CD, security | VPC + bastion + private instances |
